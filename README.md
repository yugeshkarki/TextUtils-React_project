# 📝 TextUtils - React Text Utility App

TextUtils is a simple and modern React-based web application that allows users to perform various text transformations and analysis such as converting case, counting words/characters, clearing text, and estimating reading time.

---

## 🚀 Features

- 🔠 Convert text to UPPERCASE  
- 🔡 Convert text to lowercase  
- 🧹 Clear text instantly  
- 📊 Word and character count  
- ⏱️ Estimated reading time  
- 👀 Live text preview  
- 🌙 Dark/Light mode toggle  
- 🔔 Alert notifications for actions  
- 📱 Fully responsive UI using Bootstrap  

---

## 🛠️ Tech Stack

- React.js (Functional Components + Hooks)  
- JavaScript (ES6+)  
- Bootstrap 5  
- HTML5 & CSS3

## ⚙️ Installation & Setup

cd textutils  

### Install dependencies
npm install  

### Run the project
npm start  

---
## 🎯 How It Works

- User enters text in textarea  
- Click buttons to perform actions:
  - Convert to uppercase  
  - Convert to lowercase  
  - Clear text  
- React state updates instantly using Hooks  
- Alerts show feedback for actions  
- Dark mode changes UI dynamically
